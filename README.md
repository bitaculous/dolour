[Dolour]
========

[![Gemnasium Status][Gemnasium Status]][Gemnasium]

**Extract the dominate colors from image.**

Install
-------

    $ git clone git@github.com:bitaculous/dolour.git dolour
    $ cd dolour
    $ bundle
